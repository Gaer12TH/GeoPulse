## ğŸ“¦ Installation

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Quick Start

```bash
# Clone the repository
git clone https://github.com/Gaer12TH/GeoPulse.git

# Navigate to project directory
cd GeoPulse

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`
